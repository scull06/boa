pub struct Sym {
    pub(crate) value: NonZeroUsize,
}
Expand description

The string symbol type for Boa.

This symbol type is internally a NonZeroUsize, which makes it pointer-width in size and it’s optimized so that it can occupy 1 pointer width even in an Option type.
